# Project 5 – Microsoft Entra ID Security & Identity Protection

## Overview

This project demonstrates identity and access management (IAM) administration using Microsoft Entra ID, focusing on MFA, Conditional Access, RBAC, and identity security controls.

## Technologies Used

- Microsoft Entra ID
- Microsoft 365
- Conditional Access
- Multi-Factor Authentication (MFA)
- Role-Based Access Control (RBAC)

## Key Activities

- Configured Multi-Factor Authentication (MFA)
- Implemented Conditional Access policies
- Managed users, groups, and administrative roles
- Monitored sign-in activity and authentication events
- Applied least-privilege access principles
- Reviewed identity security recommendations
- Investigated authentication anomalies

## Skills Demonstrated

- Identity and Access Management (IAM)
- Microsoft Entra ID Administration
- Conditional Access
- MFA Implementation
- RBAC
- Identity Protection
- Access Governance

## Security Controls

- Multi-Factor Authentication
- Conditional Access Policies
- Least Privilege
- Administrative Role Management
- Identity Monitoring

## Outcome

Successfully implemented identity security controls and strengthened authentication security through MFA, Conditional Access, and access governance best practices.
