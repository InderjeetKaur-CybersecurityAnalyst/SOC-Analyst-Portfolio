# Project 1 – Microsoft Defender XDR SOC Lab

## Overview

This project demonstrates the deployment and use of Microsoft Defender XDR within a Security Operations Center (SOC) lab environment.

The objective was to simulate real-world security monitoring, incident investigation, threat hunting, and endpoint detection and response activities.

## Technologies Used

- Microsoft Defender XDR
- Microsoft Defender for Endpoint
- Windows 11
- Windows Server 2022
- Active Directory
- Microsoft Entra ID
- KQL (Advanced Hunting)

## Key Activities

- Onboarded Windows endpoints into Microsoft Defender XDR
- Investigated alerts and incidents
- Conducted threat hunting using Advanced Hunting (KQL)
- Performed EICAR malware simulation testing
- Analyzed security events and endpoint telemetry
- Validated detection and response capabilities

## Skills Demonstrated

- Security Monitoring
- Threat Hunting
- Incident Response
- Endpoint Detection and Response (EDR)
- Log Analysis
- KQL Querying
- Security Investigation

## MITRE ATT&CK Techniques

- T1059 – Command and Scripting Interpreter
- T1110 – Brute Force
- T1078 – Valid Accounts
- T1021 – Remote Services

## Outcome

Successfully simulated enterprise security operations using Microsoft Defender XDR and developed hands-on experience investigating alerts, analyzing telemetry, and responding to security incidents.
