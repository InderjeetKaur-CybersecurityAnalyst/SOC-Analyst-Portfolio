# Project 2 – Active Directory Security Administration & Hardening

## Overview

This project demonstrates hands-on administration and security hardening of an Active Directory environment using Windows Server 2022.

The objective was to implement identity security controls, user and group management, Group Policy configuration, and least-privilege access principles.

## Technologies Used

- Windows Server 2022
- Active Directory Domain Services (AD DS)
- Group Policy Management
- Microsoft Entra ID
- Windows 11

## Key Activities

- Installed and configured Active Directory Domain Services
- Created and managed Organizational Units (OUs)
- Created and managed users and security groups
- Applied Group Policy Objects (GPOs)
- Implemented password and account lockout policies
- Configured role-based access control (RBAC)
- Applied least-privilege access principles
- Integrated identity security best practices

## Skills Demonstrated

- Active Directory Administration
- Identity and Access Management (IAM)
- Group Policy Management
- RBAC
- Least Privilege
- User Lifecycle Management
- Security Hardening

## Security Controls

- Multi-Factor Authentication (MFA)
- Account Lockout Policy
- Password Complexity Enforcement
- Privileged Access Management Principles

## Outcome

Successfully built and secured an Active Directory environment while applying enterprise identity security and access management best practices.
